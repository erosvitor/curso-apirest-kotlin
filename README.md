## Sobre
O projeto contém tópicos sobre o desenvolvimento de APIs REST utilizando Kotlin, Ktor e Kodein.

## Tópicos
* Banco de dados
* Serialização
* Injeção de dependências
* Arquivo de propriedades
* Tratamento de exceções
* Log
